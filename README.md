This package is related to the publication: Muller, Efrat, Itamar Shiryan, and Elhanan Borenstein. __"Multi-omic integration of microbiome data for identifying disease-associated modules."__ _Nature Communications_ 15.1 (2024): 2621. [Link](https://www.nature.com/articles/s41467-024-46888-3)

### Installation

MintTea can be installed directly from GitHub, by running the following:

```
install.packages(devtools)  
library(devtools)   
install_github("efratmuller/MintTea")   
library(MintTea)
```
